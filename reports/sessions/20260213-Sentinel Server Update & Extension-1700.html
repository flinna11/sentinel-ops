    • Incremental Setup: Configure Promtail to scrape only standard container logs first.

  • Phase VIII: The Command Center (Mattermost)

    • Deploy Mattermost (Pod or LXC). Install the desktop app on Windows 11.

  • Phase IX: GitOps Governance (ArgoCD)

    • Deploy ArgoCD. Link it to your GitHub repo.

  • Phase X: Production-Grade Messaging (RabbitMQ HA)

    • Deploy HA RabbitMQ via ArgoCD. Configure 40Gi replicated volumes and 1Gi memory limits.
